Glosario

A | B | C | D | E | F | G | H | I | J | K | L | M

A

A – Algoritmo Conjunto finito de instrucciones o pasos definidos que permiten resolver un problema o realizar una tarea específica en programación.

B

B – Backend Parte del software que gestiona la lógica, bases de datos y procesamiento en un sistema, y que no es visible directamente para el usuario.

C

C – Compilador Programa que traduce el código fuente escrito en un lenguaje de programación a un lenguaje de máquina ejecutable por un computador.

D

D – Depuración (Debugging) Proceso de identificar, analizar y corregir errores o fallos en el código de un programa.

E

E – Entorno de Desarrollo Integrado (IDE) Aplicación que ofrece herramientas como editor de código, depurador y compilador en un solo entorno para facilitar la programación.

F

F – Framework Conjunto de herramientas, bibliotecas y convenciones que simplifican el desarrollo de aplicaciones de software siguiendo una arquitectura determinada.

G

G – Git Sistema de control de versiones distribuido que permite a los desarrolladores registrar cambios en el código y trabajar de manera colaborativa.

H

H – HTML (HyperText Markup Language) Lenguaje de marcado utilizado para estructurar contenido en la web.

I

I – Interfaz de Usuario (UI) Conjunto de elementos visuales con los que un usuario interactúa dentro de una aplicación o sistema.

J

J – Java Lenguaje de programación orientado a objetos, ampliamente usado para desarrollo web, móvil y empresarial.

K

K – Kubernetes Plataforma de código abierto para automatizar el despliegue, escalado y gestión de aplicaciones en contenedores.

L

L – Librería (Library) Colección de funciones o recursos reutilizables que pueden ser invocados por programas para facilitar ciertas tareas.

M

M – Middleware Software que actúa como puente entre el sistema operativo y las aplicaciones, permitiendo su comunicación.
